Riding Cities – Intégration HTML / CSS

Création de sections HTML + intégration fidèle selon la maquette Figma

Objectif du projet

Le projet Riding Cities a pour objectif d’apprendre les bases de l’intégration HTML et CSS à partir d’une maquette Figma. J’ai dû analyser une structure HTML existante, créer une nouvelle section, modifier un bouton, ajouter une deuxième version de celui-ci, appliquer les classes CSS fournies et intégrer les documents PDF de téléchargement.

Technologies utilisées

  • HTML5
  • CSS3 (classes existantes, sans modification du fichier CSS)
  • Maquette Figma
  • Structure en colonnes
  • Attributs HTML (href, alt, src…)

Travail réalisé

Action 1 – Ajout du titre et de la couleur de fond

• Ajout du titre “Découvrez nos cours” en suivant la structure des autres sections.
• Application de la classe CSS existante permettant un fond gris.
• Respect de la hiérarchie H2/H3 déjà présente dans le site.

Action 2 – Modification du bouton existant

• Application des classes CSS pour obtenir un bouton arrondi, coloré et correspondant à la maquette.
• Ajout d’un effet hover grâce aux classes CSS fournies.
• Le bouton “Télécharger le planning de cours” a ainsi été modernisé.

Action 3 – Ajout du deuxième bouton et mise en forme en colonne

• Duplication du premier bouton pour en créer un deuxième.
• Modification de la classe du parent pour placer les deux boutons en colonne.
• Mise à jour du texte :
  – “Planning enfants”
  – “Planning adultes”
• Application des bonnes couleurs (une couleur par bouton).

Action 4 – Rattachement des documents PDF

• Analyse du dossier “Documents”.
• Attribution des bons fichiers PDF via l’attribut href.
• Le premier bouton mène au PDF Enfants, le second au PDF Adultes.
• Vérification du bon fonctionnement du téléchargement.

Ce que j’ai appris

Ce projet m’a permis de :
• comprendre comment adapter un HTML existant à une nouvelle maquette ;
• utiliser des classes CSS déjà créées sans modifier le fichier CSS ;
• travailler la structure HTML en colonnes ;
• intégrer correctement des images et leurs attributs alt ;
• créer des liens de téléchargement fonctionnels ;
• améliorer la cohérence visuelle avec Figma comme référence ;
• appliquer des bonnes pratiques d’intégration front-end.
