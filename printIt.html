Print It – Carousel Dynamique

Développement d’un carrousel dynamique en JavaScript

Objectif du projet

L’objectif de ce projet était de créer un carrousel d’images entièrement dynamique en JavaScript vanilla. Le site disposait déjà d’une structure HTML et CSS, et j’ai dû y intégrer des éléments interactifs comme les flèches de navigation, les bullet points, ainsi que la gestion automatique du changement d’image. La logique du carrousel a ensuite été rendue circulaire afin d’assurer une navigation fluide et continue.

Technologies utilisées

  • JavaScript Vanilla
  • DOM Manipulation
  • Event Listeners & Navigation
  • CSS Positioning
  • Debugging & Console Testing

Ce que j’ai appris

Ce projet m’a permis de comprendre en profondeur le fonctionnement du DOM et les interactions JavaScript. J’ai travaillé sur :

• l’ajout et la gestion d’Event Listeners,
• la création et la mise à jour dynamique des bullet points,
• la navigation circulaire dans un tableau d’images,
• la mise en place d’un système de slides modulaire,
• ainsi que le débogage via la console pour vérifier les comportements.

J’ai également réalisé des tests UI & API avec Cypress pour valider le bon fonctionnement du carrousel et des interactions utilisateur.
